# ShopTracker

A simple, mobile-friendly shopping list calculator that makes it easy to track items and total costs while shopping.

## Features (Phase 1)
- Add/remove items with price and quantity
- Real-time calculation of subtotal, tax, and total
- Mobile-friendly interface
- No installation required - runs in browser

## Development Setup
1. Clone the repository
2. Open `index.html` in your browser
3. Start adding items to your shopping list

## Testing Locally
Simply open `index.html` in your web browser. No server required for basic functionality.
